🐙 رفع المشروع على GitHub

1
اذهب إلى github.com وأنشئ حساباً مجانياً إذا لم يكن لديك
2
انقر + ← New repository ← اكتب اسم مثل nexus-trade ← Create repository
3
انقر uploading an existing file ← اسحب جميع الملفات ← Commit changes
4
لنشر الموقع: Settings ← Pages ← اختر main ← Save
رابطك سيكون: https://USERNAME.github.io/nexus-trade/nexus-trade-full.html

💻 باستخدام Git (أسرع للتحديثات)

1
حمّل Git من git-scm.com وثبّته بالإعدادات الافتراضية
2
افتح CMD داخل مجلد المشروع (Shift + كليك يمين ← Open PowerShell here)
3
نفّذ هذه الأوامر:
# استبدل USERNAME باسم حساب GitHub الخاص بك
git init
git add .
git commit -m "Initial: NEXUS TRADE v2.0"
git branch -M main
git remote add origin https://github.com/USERNAME/nexus-trade.git
git push -u origin main
تحديث الملفات لاحقاً: بعد أي تعديل نفّذ: git add . && git commit -m "Update" && git push
